## Overview

This project presents a hybrid recommendation system designed to assist students in finding suitable European universities according to their academic preferences.

The system combines Natural Language Processing (NLP), Machine Learning, and personalized filtering techniques to generate relevant university recommendations.

## Features

- AI-based university recommendations
- Semantic similarity using Sentence-BERT
- University clustering using K-Means
- Cosine similarity ranking
- Filtering based on:
  - Country
  - Field of study
  - Tuition fees
  - Scholarships
  - University ranking

## Methodology

The system pipeline:

1. Data collection and preprocessing
2. Feature engineering
3. Sentence-BERT embedding generation
4. K-Means clustering
5. Similarity-based recommendation
6. Personalized ranking
